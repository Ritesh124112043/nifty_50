# Nifty 50 Analytics Dashboard 📈

A machine learning-powered web application built with Python and Streamlit to analyze and predict Nifty 50 stock trends using a Random Forest model.

🚀 **Live Demo:** [Click here to view the live dashboard!](https://huggingface.co/spaces/Ritesh124112043/nifty_50)

### Features
* Real-time adjustment of Machine Learning parameters (RF Estimators, Test Size).
* Interactive data visualization using Plotly.
* Feature importance and predictive analytics.

### Tech Stack
* Python
* Streamlit
* Scikit-Learn
* Pandas & Numpy
* Plotly
