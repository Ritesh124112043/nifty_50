Here's your Streamlit dashboard! Here's how to run it:

Install dependencies (if needed):

bash
pip install streamlit plotly scikit-learn openpyxl

Run the app:

bash
streamlit run nifty50_dashboard.py

What's inside — 5 tabs:

  • 📊 Price Overview — Interactive price + MA10 trend chart, return distribution, and volatility over time
  • 🔬 Feature Analysis — Correlation heatmap and interactive scatter plots for each feature vs Return
  • 🤖 Model Performance — MAE / MSE / R² for train & test, actual vs predicted return chart, residuals distribution
  • 🏆 Feature Importance — Random Forest feature importance bar chart + scatter matrix
  • 🔮 Prediction — Enter custom values in the sidebar and get a live predicted return

Sidebar controls:

  • Upload your Nifty 50.xlsx file (it runs on synthetic demo data until you do)
  • Adjust RF estimators and test split size
  • Enter price/lag/volatility values for on-demand prediction
